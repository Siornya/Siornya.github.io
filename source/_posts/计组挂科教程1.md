## 计算机系统简介

计算机由**硬件**和**软件**组成。硬件指实体部分，软件指具有各类特殊功能的程序。

计算机硬件包括存储器，系统总线，CPU和I/O设备

*CPU = 运算器(ALU) + 控制器(CU)*

## 计算机的基本组成

**冯诺依曼结构**

1. 计算机由五大部件组成

2. 指令和数据以同等地位存于存储器，可直接按地址寻访

3. 指令和数据用二进制表示

4. 指令由操作码和地址码组成

5. 存储程序

6. 以运算器为中心

**主存储器的基本组成**

主存储器由**存储体**、**MAR**（存储器地址寄存器）、**MDR**（存储器数据寄存器）组成。

- 存储体：由若干**存储单元**组成，每个存储单元存放一个**存储字**
- MAR：存放要访问的存储单元**地址**，位数反映了最多可寻址的存储单元个数，4位MAR表示有16个存储单元
- MDR：存放从存储体读出或写入的数据，位数等于存储字长

**运算器的基本组成**

- **ACC**（累加器）：存放操作数和运算结果
- **MQ**（乘商寄存器）：乘除运算时存放操作数或结果
- **X**（操作数寄存器）：存放操作数
- **ALU**：算术逻辑运算单元

**控制器的基本组成**

- **PC**（程序计数器）：存放当前指令地址，自动加 1 指向下一条指令
- **IR**（指令寄存器）：存放当前正在执行的指令
- **CU**（控制单元）：分析指令，发出控制信号

## 计算机硬件的主要技术指标

1. 机器字长 CPU一次能处理的数据位数，与CPU中的寄存器位数有关

2. 运算速度 

    | 指标 | 说明 |
    |------|------|
    | **主频**（Clock Rate） | CPU 的时钟频率，单位 Hz |
    | **吉普森法** | 各类指令的执行时间加权平均 |
    | **MIPS** | 每秒执行百万条指令数 |
    | **CPI**（Clocks Per Instruction） | 执行一条指令所需的时钟周期数 |
    | **FLOPS** | 每秒浮点运算次数（百万/十亿/万亿） |
    
3. 存储容量

## 小结

| 要点 | 内容 |
|------|------|
| 冯诺依曼特征 | 存储程序，二进制，以运算器为中心 |
| 五大部件 | 运算器、控制器、存储器、输入、输出 |
| CPU 组成 | 运算器 + 控制器（ALU + CU）|
| 关键寄存器 | PC、IR、ACC、MAR、MDR |
